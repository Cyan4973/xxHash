Mixes all bits to finalize the hash.

The final mix ensures that all input bits have a chance to impact any bit in the output digest, resulting in an unbiased distribution.

static

Processes the last 0-31 bytes of ptr.

There may be up to 31 bytes remaining to consume from the input. This final stage will digest them to ensure that all input bytes are present in the final mix.

Parameters
hashThe hash to finalize.
ptrThe pointer to the remaining input.
lenThe remaining length, modulo 32.
alignWhether ptr is aligned.
Returns
The finalized hash
